# -Lab2_C_Doaga_Ion
Scopul lucrării: Studierea posibilităţilor şi mijloacelor limbajului C pentru programarea algoritmilor cu structură ramificată şi ciclică la prelucrarea tablourilor unidimensionale.
In acest program se introduc un numar de elemente in care se determina numarul maxim negativa si pozitia in care acesta se afla si deasemenea sa se afle cate numere negative sunt.
gcc main.c -o program
Introdu numarul de elemente dorit: 4
Introdu elementul: -3
4
6
-2
Elementele sunt: -3, 4, 6, -2,
Numarul maxim negativ este -2 si pozitia lui este 3.
Numere negative in tablou sunt: 2
